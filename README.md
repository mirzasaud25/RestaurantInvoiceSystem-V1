#RestaurantInvoiceSystem-V1

This is a simple Java console application to manage restaurant billing
You can create invoices, add items, calculate the total bill, make payments, and print receipts.

Features
Login as cashier
Select items from the menu
Generate invoice and calculate total
 Accept cash payment and calculate change
Print full receipt with details

How to Run 
1. Clone the repository
2. Open in any Java IDE
3. Run SaleReceipt.java in the salereceipt package
